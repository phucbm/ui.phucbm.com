## How to Use

I publish all components following the **shadcn registry ecosystem**. If you're already using shadcn, this will feel right at home. Just grab what you need and go.

For those working on other platforms like **Webflow**, **Framer**, or similar -
you'll find an <span className="pointer-events-none inline-block translate-y-2"><OpenInV0Button/></span> button on each component. This deploys a live example in v0 where you can ask AI to customize it to fit your exact needs.
