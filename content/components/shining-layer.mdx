# Shining Layer

Pure CSS to add an animated shining effect that activates on hover.

<RegistryInstall name="shining-layer"/>

## Usage

The component uses CSS animations and gradient backgrounds to create a shining effect. It must be used inside an element with the `group/shining` class and `overflow-hidden relative` positioning.

```text
group/shining overflow-hidden relative
```

```tsx
<a href="#"
   className="group/shining overflow-hidden relative inline-flex items-center justify-center"
>
  <span>Hover me</span>
  <ShiningLayer />
</a>
```
