# Props

### SlideCarousel Component

| **Prop** | **Type** | **Default** | **Description** |
|-----------|-----------|--------------|-----------------|
| `baseSpeed` | `number` | `1` | Controls how fast the carousel moves automatically. Higher values move faster. |
| `hoverSlowdownRatio` | `number` | `0.5` | Multiplier applied when hovering. `1` keeps speed constant, lower values slow down, higher values speed up. |
| `autoSlide` | `boolean` | `true` | Enables or disables continuous automatic scrolling. |
| `infinite` | `boolean` | `true` | Determines whether the carousel loops infinitely. If `false`, scrolling stops at edges. |
| `manualNav` | `boolean` | `false` | Toggles manual navigation controls (left/right arrows). |
| `images` | `string[]` | Internal default array | Optional list of image URLs to render. Defaults to images inside `/public/FadingTransition/medias/`. |
